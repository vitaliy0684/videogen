ОШИБКА API — ПОШАГОВОЕ РЕШЕНИЕ

1) Проверка ключа
- Убедись, что API_KEY заполнен в config.py/.env.
- Ключ не должен содержать "ваш_ключ" или пробелы.

2) Проверка баланса
- Открой личный кабинет proxyapi.ru.
- Проверь, что есть активный баланс/лимиты.

3) Альтернативные действия
- Попробуй другую модель (например, замени VIDEO_MODEL).
- Проверь PROXY_API_URL (по умолчанию: https://api.proxyapi.ru/openai/v1).

4) Тестовый запрос через curl
curl -X GET "https://api.proxyapi.ru/openai/v1/models" ^
  -H "Authorization: Bearer YOUR_API_KEY"

Если запрос не проходит:
- Проверь интернет.
- Проверь firewall/proxy.
- Проверь, не истек ли ключ.
